CHANGE DIXIE
2020 — Web Design,  Development,  Personal Project
changedixie.com ↗

I started this project to spread information about my undergrad school's name, “Dixie”, and the unsavory connections it has to the Antebellum South. Additionally, the goal was to encourage Southern Utahns to talk about changing Dixie State University's name.
‍
The website exhibits newspapers, yearbooks and articles that connect the university to the Confederacy. Growing up in St.George, I found that much of this information wasn't widely known. I wanted the project to be firm, but not overbearing, and prompt the reader to think about these important issues.
This project was chosen by AIGA SLC to be displayed in The 100 Show 2020.

Dixie State University has since been renamed Utah Tech University.
