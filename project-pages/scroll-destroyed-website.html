Scroll Destroyed Website

Personal Project

I enjoy experimenting with code and new digital tools. Creative tech is really compelling.

For this project, I created a website that destroys its content as the user scrolls down. To achieve this effect, I used Lottie, a web animation tool that allows After Effects animations to be implemented on the web. I admit that I pushed Lottie beyond its intended use case to see its limitations. The result was pretty fun.

Feel free to check out the live site here.
